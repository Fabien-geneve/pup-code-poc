How a P2P Social Network could work




Create a new "Network of Friends"


This can be split in 2 cases :

  1. The PUP Code application is already installed on one or your device (Smartphone, Tablet, Laptop ...).
  2. The PUP Code application is not yet installed on one of your devices.
This mechanism ensure that each User or Peer, have a correct network configuration, and could make a direct TCP/IP connection on the other network members, without transiting on a third party server on which your data could be stored"
In fact, these cases are the same ! Just the first case, you need to install the PUP Code application, at least on one of your devices. See the PBM diagram above, it illustrate how you could create a new "Network Friends", when the PUP Code application is already installed.

Accept an invitation to join a "Friends Network"


First, the PUP Code service receive an invitation to Join a "Friends Network", sent by the coordination server. Then, when accepted, the application notify the coordination server of the user acceptance, and the coordination server sent the new network configuration of each Peer of the system. Each User or Peer in the System, need to accept to joint the "Friends Network".
This mechanism ensure that each User or Peer, have a correct network configuration, and could make a direct TCP/IP connection on the other network members, without transiting on a third party server on which your data could be stored.

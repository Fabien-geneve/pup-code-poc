Comment un "Réseau Social" P2P pourrait fonctionner


Création d'un nouveau réseau d'amis

Deux cas peuvent se présenter :

1. L'application PUP Code est installée sur l'un de mes outils (Smartphone, Tablet, Laptop ...).

2. L'application PUP Code n'est pas installée sur aucun de mes appareil.

En fait ce sont les même cas d'utilisation : le cas 2 représente le cas ou il faut, au préalable, installer l'application PUP Code sur au moins un de ses appareil. Aussi le diagramme montre comment la création du réseau, pourrait se faire avec une application déjà installée.


Accepter une invitation à rejoindre un "Réseau d'amis"


Pour accepter une invitation, l'application reçoit une notification du serveur de coordination : lorsque l'utilisateur accepte l'invitation, l'application envoie un message de "création de nouveaux réseaux" aux autres membres connus du réseau. Alors chaque utilisateur, dois confirmer qui veut bien rejoindre le réseau.

Une fois la confirmation effectuée, un message est envoyé au serveur de coordination. Ensuite ce serveur notifie chaque membre du réseau (chaque "Peer" en fait) en lui envoyant la nouvelle configuration du "réseau d'amis".

Ceci permettra, à chaque utilisateur du réseau d'avoir une configuration toujours correcte, et ainsi de faire directement des connexions TCP/IP aux autres membres du réseau, sans passer par un serveur tiers ou les données seraient stockées.
